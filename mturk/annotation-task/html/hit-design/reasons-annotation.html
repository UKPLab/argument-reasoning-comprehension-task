Task

There are 5 arguments in total in this HIT that are completely independent of each other. Each deals with a different topic. For each argument there are two tasks to be done.

First, choose author's overall standpoint towards the topic

  • Read the argument fist, make sure you understand it. Choose from the two opposing options offered. If neither of them apply (for example, if the argument does not even talk about the topic), choose the last option ("The argument does not take a stance to the topic").

Second, find and mark author's standpoint in text, if it is explicitly written

  • If the author takes stance on the topic, can you find whether the standpoint is mentioned in the text explicitly? If so, mark sentence or sentences that directly say so by clicking on them.
    • By explicit standpoint we mean that the sentence (or sentences) must be understandable on its own and it must clearly say what is the author's position.
    • It is usually done by rephrasing the debate topic (for example: "Yes, cloning should be definitely abandoned").
    • Using pronouns is also allowed (for example: "He was the best singer ever!" for the debate title "Was Frank Sinatra the best singer?")"
    • Reasons explaining why the standpoint is taken, must not be marked. (for example: "He could sing anything!" is not the main standpoint for the debate title "Was Frank Sinatra the best singer?")"
  • If the author's stance is not explicit (but you can infer it with your common sense), check the option "I couldn't find author's standpoint explicitly written in the argument."
